# 📋 Инструкция: Как увидеть новую модалку принятия заказа

## 🎯 Что было реализовано

Создана улучшенная модалка принятия производственных заказов с тремя этапами:

### ✅ Этап 1: Проверка заказа
- Просмотр деталей заказа (клиент, объём, сроки, состав)
- Подтверждение доступности материалов
- Подтверждение реалистичности сроков
- Выбор: принять или отклонить заказ

### 💰 Этап 2: Стоимость изготовления
- Ввод цены за единицу для каждого SKU
- Автоматический расчёт итогов по позициям
- Общая стоимость заказа
- Визуальные подтверждения (зелёные галочки)

### ❌ Альтернатива: Отклонение заказа
- Выбор причины из списка
- Развёрнутый комментарий для клиента
- Прозрачная коммуникация

---

## 🚀 Как увидеть новую модалку

### Шаг 1: Перейдите в производственный портал
Откройте в браузере: `/production/orders`

### Шаг 2: Выберите новый заказ
В списке найдите заказ с **красным бейджем "Новый"**:
- **ID:** PO-2026-00041
- **Клиент:** Kamilek
- **Сезон:** Весна 2026
- **Статус:** Новый
- **Объём:** 1050 шт

### Шаг 3: Откройте заказ
Кликните на карточку заказа - вы попадёте на страницу детального просмотра: `/production/orders/PO-2026-00041`

### Шаг 4: Нажмите "Принять заказ"
В правом верхнем углу страницы есть две кнопки:
- **Запросить изменения** (белая)
- **Принять заказ** (зелёная) ← нажмите её!

### Шаг 5: Пройдите процесс принятия
1. **Проверьте заказ:** отметьте чекбоксы о материалах и сроках → Далее
2. **Укажите цены:** введите стоимость изготовления для каждого товара
   - Брюки классические палаццо: например, 1500 ₽/шт
   - Топ базовый: например, 800 ₽/шт
3. **Примите заказ:** нажмите "Принять заказ в работу"

---

## 🔄 Альтернативный путь: Отклонение

На этапе "Проверка заказа" можно нажать кнопку **"Отклонить"** (красная):
1. Выберите причину из списка
2. Напишите комментарий для клиента
3. Подтвердите отклонение
