# 🎨 Варианты подтверждения товара на Шаге 3

## 📋 Обзор

Различные подходы к UX для подтверждения найденного товара и формирования группы товаров для распределения КИЗов.

---

## ✅ Вариант 1: Текущий (Автоподбор с кнопками)

### Описание
- Система автоматически находит товар
- Показывается карточка с badge "Найден автоматически"
- Кнопки: "Сбросить автоподбор" и "Изменить товар"

### Макет
```
┌─────────────────────────────────────────────────────────┐
│ 🖼️ Брюки женские               [Найден автоматически]  │
│    Артикул: ART-001 • Цвет: Черный • Категория: Брюки   │
│    Размеры: 40  42  44  46  48  50                      │
│                                                          │
│    [Сбросить автоподбор]  [Изменить товар]              │
└─────────────────────────────────────────────────────────┘
```

### ✅ Преимущества
- Быстрый старт (товар уже найден)
- Минимум действий для пользователя
- Понятно откуда взялся товар

### ❌ Недостатки
- Что если автоподбор ошибся?
- Непонятно как "изменить товар"

---

## 🎯 Вариант 2: Выбор из списка похожих товаров

### Описание
- Система показывает топ-3 наиболее подходящих товара
- Пользователь выбирает один кликом
- Или ищет другой товар

### Макет
```
┌─────────────────────────────────────────────────────────┐
│ Выберите товар для распределения кодов                  │
│                                                          │
│ Система нашла похожие товары:                           │
│                                                          │
│ ○ 🖼️ Брюки женские                         [95% совпад.]│
│      Артикул: ART-001 • Черный • 40-50                  │
│                                                          │
│ ○ 🖼️ Брюки женские классические           [87% совпад.]│
│      Артикул: ART-003 • Черный • 42-50                  │
│                                                          │
│ ○ 🖼️ Брюки женские зауженные              [82% совпад.]│
│      Артикул: ART-004 • Черный • 40-48                  │
│                                                          │
│ [Поиск другого товара...]                               │
└─────────────────────────────────────────────────────────┘
```

### ✅ Преимущества
- Пользователь сам выбирает
- Видны альтернативы
- Процент совпадения помогает принять решение

### ❌ Недостатки
- Требуется дополнительный клик
- Что если нужного товара нет в топ-3?

## 💡 Рекомендации по выбору

### Для FASTWMS подойдет:

**Основной:** Вариант 1 (Автоподбор) + Вариант 3 (Поиск)
- Быстро для опытных пользователей
- Гибко для сложных случаев
- Понятно для новичков

**Улучшение текущего варианта:**

1. **Добавить AI-бейдж** из Варианта 6
```
🖼️ Брюки женские                    [AI 96% ✓]
```

2. **Добавить превью данных** из файла под карточкой
```
Данные из файла: Брюки женские • 529940427 • Черный
```

3. **Сделать "Изменить товар"** как Вариант 3
- Клик → открывается модалка с поиском
- Автодополнение при вводе
- Показ топ-3 похожих

4. **Добавить сравнение** при наведении
```
Навести на ℹ️ → Tooltip:
┌──────────────────────────┐
│ Совпадения:              │
│ ✓ Название: 98%          │
│ ✓ Артикул найден         │
│ ✓ Цвет совпадает         │
│ ⚠ Размеры: 4 из 6        │
└──────────────────────────┘
```
