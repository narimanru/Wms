# Инструкция: Как увидеть новую модалку принятия заказов

## Обновлённый функционал производителя

Была создана новая модалка `AcceptOrderModal` с трёхшаговым процессом:

### Шаг 1: Проверка заказа
- Просмотр деталей (клиент, объём, сроки, состав)
- Подтверждение доступности материалов
- Подтверждение реалистичности сроков
- Выбор: принять или отклонить

### Шаг 2: Стоимость изготовления
- Ввод цены за единицу для каждого SKU
- Автоматический расчёт итогов по позициям
- Общая стоимость заказа

### Шаг 3 (альтернативный): Отклонение
- Выбор причины (сроки, материалы, мощность, качество, цена, другое)
- Комментарий для клиента

## Как протестировать:

### Вариант 1: Через селектор порталов
1. Откройте `/portal-selector` в браузере
2. Нажмите на карточку **"Производственный портал"**
3. Вы попадёте в список заказов производителя
4. Откройте любой заказ со статусом **"Ожидает подтверждения"** (например, ORD-2025-003)
5. Нажмите кнопку **"Принять заказ"** в верхней части страницы
6. Откроется новая трёхшаговая модалка

### Вариант 2: Прямая ссылка
1. Откройте `/production/orders` - список заказов производителя
2. Кликните на строку заказа или кнопку "Открыть"
3. В деталях заказа нажмите "Принять заказ"

### Вариант 3: Конкретный заказ
Прямая ссылка на заказ с кнопкой: `/production/orders/ORD-2025-003`

## Важно!
Есть две разные системы:
- `/production-orders/:id` - **СТАРАЯ** страница (без новой модалки)
- `/production/orders/:id` - **НОВАЯ** производственный портал (с новой модалкой) ✅

Используйте **вторую** ссылку для тестирования!
