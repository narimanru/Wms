# 🎯 Как найти новый раздел "Импорт КИЗов v2"

## Способ 1: С главной страницы (Dashboard)

1. Откройте главную страницу FASTWMS (/)
2. В разделе **"Быстрые действия"** найдите карточку **"Импорт КИЗов v2"** с иконкой ✨ Sparkles
3. Карточка содержит описание: *"Новый wizard с автоопределением GTIN и привязкой товаров"*
4. Нажмите на карточку — откроется новый wizard импорта

## Способ 2: Прямая ссылка

Перейдите по адресу: `/import-kiz`

---

## 📋 Что вы увидите

### Новый пошаговый wizard импорта КИЗов состоит из 7 шагов:

1. **Загрузка файлов** 
   - Множественная загрузка файлов (можно загрузить сразу несколько)
   - Drag & drop зона
   - Прогресс загрузки и обработки каждого файла
   - Таблица со статусами: загружается, обрабатывается, успешно, ошибка
   - Возможность удалить или повторить обработку файла

2. **Анализ файлов** 
   - Автоматическое определение GTIN для каждого файла
   - Определение товара, цвета и размера
   - Количество найденных КИЗов
   - Уровень уверенности: высокий, средний, низкий
   - Источник данных: GTIN, PDF, название файла

3. **Подтверждение товара** 
   - Автоматическая подстановка найденного товара
   - Кнопка "Сбросить автоподбор"
   - Кнопка "Изменить товар"
   - Поиск товара с autocomplete
   - Умные предупреждения при выборе другого типа товара или цвета

4. **Проверка размеров** 
   - Проверка соответствия размера в файле и доступных размеров товара
   - Визуальные индикаторы: совпадает / есть расхождение
   - Предупреждения при несоответствии
   - Подсказки по источнику определения размера (PDF или название файла)

5. **Распределение по размерам** 
   - Сводка: всего КИЗов, распределено, осталось
   - Таблица распределения с колонками: файл, товар, цвет, размер, количество, основание выбора
   - Выбор размера из доступных (40, 42, 44, 46, 48, 50 для брюк)
   - Кнопка "+ Добавить размер"
   - Валидация суммы в реальном времени
   - Цветовые индикаторы статуса
   - Предупреждения о расхождениях

6. **Финальное подтверждение** 
   - Список всех загруженных файлов
   - Полная сводка распределения по товарам и размерам
   - Итоговое количество КИЗов к импорту

7. **Результат импорта** 
   - Успешный импорт с детальной статистикой
   - Количество обработанных файлов
   - Количество импортированных КИЗов
   - Распределение по товарам и размерам
   - Кнопки: Закрыть / История загрузок / Импортировать еще

## 🔑 Ключевые возможности

### Множественная загрузка
- Загрузка нескольких файлов одновременно
- Прогресс для каждого файла отдельно
- Возможность удалить или повторить обработку
- Поддержка PDF, CSV, XLSX

### Умное определение
- Автоматическое определение GTIN
- Поиск товара, цвета и размера
- Три уровня уверенности: высокий, средний, низкий
- Источник данных: GTIN, PDF, название файла

### Проверка соответствия
- Проверка размера в файле vs выбранный размер
- Предупреждения при несоответствии
- Визуальные индикаторы совпадений
- Подсказки по источнику определения

### Умные предупреждения
- **Высокий приоритет**: другой тип товара (брюки → футболка)
- **Средний приоритет**: другой цвет (черный → бежевый)
- **Средний приоритет**: несоответствие размера (48 в файле, выбран 50)
- Модальные окна с подтверждением для рискованных действий

### Гибкое распределение
- Распределение по конкретным размерам
- Можно добавить дополнительные размеры
- Валидация суммы КИЗов
- Показ основания выбора (автоматически, вручную, по GTIN, по PDF, по названию файла)

### Автоматическая привязка
- Сохранение привязок GTIN → товар
- Автоподстановка товара при повторной загрузке
- Возможность сбросить автоподбор
- Ручной выбор товара когда нужно

## 💡 Советы по использованию

### Загрузка файлов
1. **Несколько файлов одновременно** - можно выбрать сразу все файлы для загрузки
2. **Drag & Drop** - просто перетащите файлы в зону загрузки
3. **Проверка статуса** - следите за прогрессом каждого файла
4. **Повтор при ошибке** - используйте кнопку повтора если файл не обработался

### Работа с товарами
1. **Автоподбор** - система сама найдет товар по GTIN
2. **Сброс автоподбора** - если система ошиблась, сбросьте и выберите вручную
3. **Изменение товара** - можно изменить автоматически выбранный товар
4. **Предупреждения** - внимательно читайте предупреждения при выборе другого типа товара

### Проверка размеров
1. **Смотрите на индикаторы** - зеленый = совпадает, желтый = расхождение
2. **Читайте подсказки** - система объяснит, откуда взят размер
3. **Проверяйте соответствие** - особенно когда размер определен по названию файла

### Распределение
1. **Следите за балансом** - всего / распределено / осталось
2. **Используйте дропдауны** - выбирайте размер из списка
3. **Добавляйте строки** - кнопка "+ Добавить размер" для дополнительных размеров
4. **Проверяйте основание** - бейджи показывают, как был определен товар/размер

### Финальная проверка
1. **Шаг 6** - внимательно просмотрите всю сводку
2. **Проверьте итоги** - убедитесь что все КИЗы распределены правильно
3. **История** - после импорта можно открыть историю загрузок

## 📊 Пример реального сценария

### Сценарий: Загрузка КИЗов для брюк женских

**Исходные данные:**
- Товар: Брюки женские
- Цвет: Черный
- Размеры: 40, 42, 44, 46, 48, 50

**Файлы:**
```
bryuki_black_40.pdf  → 85 КИЗ, размер 40
bryuki_black_42.pdf  → 92 КИЗ, размер 42
bryuki_black_44.csv  → 110 КИЗ, размер 44
bryuki_black_48.pdf  → 120 КИЗ, размер 48
```

**Процесс:**

1. **Шаг 1**: Загружаем все 4 файла одновременно
   - Видим прогресс загрузки каждого
   - Все файлы успешно обработаны

2. **Шаг 2**: Система анализирует
   - Определяет GTIN в каждом файле
   - Находит товар "Брюки женские"
   - Определяет цвет "Черный"
   - Извлекает размеры из PDF/названий файлов
   - Считает КИЗы: 85, 92, 110, 120

3. **Шаг 3**: Подтверждение товара
   - Товар найден автоматически ✅
   - "Брюки женские, Черный" подставлен
   - Подтверждаем или меняем при необходимости

4. **Шаг 4**: Проверка размеров
   - Размер 40 найден в PDF → совпадает ✅
   - Размер 42 найден в PDF → совпадает ✅
   - Размер 44 определен по названию → проверить ⚠️
   - Размер 48 найден в PDF → совпадает ✅

5. **Шаг 5**: Распределение
   ```
   Файл                 | Размер | КИЗ | Основание
   ---------------------|--------|-----|------------
   bryuki_black_40.pdf  | 40     | 85  | По PDF
   bryuki_black_42.pdf  | 42     | 92  | По PDF
   bryuki_black_44.csv  | 44     | 110 | По названию файла
   bryuki_black_48.pdf  | 48     | 120 | По PDF
   ---------------------|--------|-----|------------
   ИТОГО:                        | 407 | 
   ```

6. **Шаг 6**: Финальная проверка
   - Проверяем все данные
   - Всего к импорту: 407 КИЗ
   - Распределено по 4 размерам

7. **Шаг 7**: Успешный импорт! 🎉
