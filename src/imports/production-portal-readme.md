# Производственный портал FASTWMS

## Обзор

Производственный портал - это отдельный кабинет для фабрик/подрядчиков, который позволяет им управлять производственными заказами, этапами производства, отгрузками и документами.

## Структура

### Маршруты

**Производственный портал (без основного Layout):**
- `/production/orders` - Список заказов
- `/production/orders/:id` - Детали заказа с табами
- `/production/calendar` - Календарь производства
- `/production/shipments` - Отгрузки
- `/production/documents` - Документы
- `/production/settings` - Настройки

**Селлер портал (с основным Layout):**
- `/production-orders` - Список заказов (вид селлера)
- `/production-orders/:id` - Детали заказа (вид селлера)

### Компоненты

1. **ProductionLayout** (`/src/app/components/ProductionLayout.tsx`)
   - Sidebar с навигацией
   - Поиск по заказам
   - Уведомления
   - Информация о подрядчике

2. **ProductionOrders** (`/src/app/pages/production/ProductionOrders.tsx`)
   - Список заказов с фильтрами
   - KPI карточки (новые, в работе, просроченные)
   - Таблица заказов

3. **ProductionOrderDetailPage** (`/src/app/pages/production/ProductionOrderDetailPage.tsx`)
   - Табы: Обзор, Состав заказа, Этапы, Оплата, Инциденты, Отгрузка
   - Модальные окна: Принять заказ, Запросить изменения, Создать инцидент

4. **ProductionCalendar** (`/src/app/pages/production/ProductionCalendar.tsx`)
   - Недельный/месячный вид
   - Заказы по срокам
   - Просроченные заказы

5. **ProductionShipments** (`/src/app/pages/production/ProductionShipments.tsx`)
   - Готовые к отгрузке заказы
   - Чек-листы готовности
   - Формирование документов

6. **ProductionDocuments** (`/src/app/pages/production/ProductionDocuments.tsx`)
   - Управление документами
   - Фильтры по типу и заказу
   - Загрузка/скачивание

### Демо-данные

Файл: `/src/app/data/production-orders.json`

Содержит:
- 2 демо-заказа (новый и в работе)
- Полная структура items с размерами
- Workflow этапов производства
- Статусы оплат
- Инциденты

## Основные функции

### Для подрядчика (Production Portal)

1. **Просмотр заказов**
   - Фильтры по статусу
   - Только срочные
   - Поиск

2. **Работа с заказом**
   - Принять/отклонить заказ
   - Ввод плана vs факта по размерам
   - Указание причин отклонений
   - Управление этапами производства

3. **Инциденты**
   - Создание инцидента (ткань, брак, сроки и т.д.)
   - Предложение решения
   - Отправка клиенту

4. **Отгрузка**
   - Чек-лист готовности
   - Загрузка документов
   - Фото коробов
   - Формирование накладных

### Для селлера

1. **Контроль производства**
   - Просмотр всех заказов
   - Утверждение отклонений
   - Контроль этапов

2. **Финансы**
   - Контроль оплат
   - Выставление счетов

## Доступ к порталам

Страница выбора портала: `/portal-selector`

Позволяет переключаться между:
- Портал селлера (основная система FASTWMS)
- Портал производства (кабинет подрядчика)

## Безопасность

КИЗы отображаются в маскированном виде: `••••1234`

## Роли и права (RBAC)

### Production Contractor (Фабрика)

**Доступно:**
- Просмотр только своих заказов
- Принять/отклонить заказ
- Изменять статусы этапов
- Вносить факт выпуска
- Создавать инциденты
- Загружать документы
- Видеть статус оплат

**Недоступно:**
- Редактирование карточек товара
- Управление КИЗ
- Доступ к складам/аналитике
- Изменение цены/условий

## Технические детали

- React Router для маршрутизации (react-router, не react-router-dom)
- TypeScript для типизации
- Inline стили для быстрой разработки
- Lucide React для иконок
- Демо-данные в JSON формате
