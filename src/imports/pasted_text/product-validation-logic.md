Шаг 3 — Подтверждение товара (предупреждение о несоответствии)

Проблема

Сейчас модальное окно появляется каждый раз при выборе товара и мешает работе.

Пользователь не может выбрать товар.

⸻

Правильная логика

⚠️ Предупреждение должно появляться только при переходе на следующий шаг, а не при клике по товару.

Пользователь выбирает товар.

Никаких модальных окон.

Очень важная доработка

Сейчас окно появляется на каждом клике.

Нужно изменить триггер.

Было

onProductSelect()
→ showWarningModal()


⸻

Должно быть

onNextStep()
→ validateProductMatch()
→ showWarningModal()


⸻
