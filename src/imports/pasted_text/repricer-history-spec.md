Ниже готовое ТЗ для Figma для блока История действий репрайсера в системе FASTWMS (в стиле Vuexy / современный SaaS-интерфейс).

⸻

ТЗ для Figma

Раздел: История действий репрайсера

Экран

Настройки → Репрайсер → Шаг 3: История действий

Назначение блока:
Показать историю изменений цен / скидок, которые были применены системой репрайсера или вручную.

История должна позволять быстро понять:
	•	когда произошло изменение
	•	какой товар изменился
	•	какая скидка применена
	•	успешно ли прошло действие
	•	по какой причине произошло изменение

⸻

2. Действие

Показывает что произошло.

Формат:

или

Цена → 1890 ₽

3. Статус

Бейдж:

Ошибка — красный
