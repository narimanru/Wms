Отличная идея — разделить товары на группы прямо в окне выбора. Это сильно упростит UX. Ниже готовое ТЗ для Figma, как это сделать.

⸻

ТЗ для Figma Make

Модальное окно: Добавить товар (группировка товаров)

Цель

Показать товары сразу сгруппированными, чтобы пользователь видел:
	•	какие товары относятся к одной модели
	•	какой цвет
	•	какая категория
	•	какие размеры есть

И мог добавлять группу товаров, а не искать каждый товар отдельно.

⸻

Заголовок модального окна

Добавить товар

Подзаголовок:

Выберите товар или группу связанных товаров для распределения кодов маркировки.

Размеры

Показывать в виде chips

Пример:

40 42 44

Если размеров больше:

40 42 44 +2

⸻

Кнопка

Кнопка справа

Добавить

Добавляет этот товар в группу импорта.

⸻

Возможность добавить всю группу

В заголовке группы добавить кнопку:

Добавить группу

Тогда добавляются все товары группы.

⸻

Ограничения

Если товар не подходит по условиям:
	•	категория отличается
	•	группа несовместима

кнопка Добавить становится disabled.

И появляется tooltip:

Товар не может быть добавлен в эту группу.

⸻

Визуальное разделение групп

Каждая группа:
	•	card container
	•	light background
	•	расстояние между группами 16–24px

⸻

Цветовая логика

Группа совпадает по цвету с основным товаром:

зелёный индикатор

Группа с другим цветом:

жёлтый индикатор
badge другой цвет

UX преимущества

Такой интерфейс позволяет:
	•	сразу увидеть все связанные товары
