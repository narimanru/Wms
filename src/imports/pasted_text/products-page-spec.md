ТЗ для Figma Make — страница «Товары»

Создать страницу списка товаров для FASTWMS в современном минималистичном стиле: светлый интерфейс, аккуратные таблицы, мягкие тени, скругления, чистая типографика, как у OpenAI/SaaS dashboard.

1. Общая структура страницы

Сделать страницу каталога товаров с:
	•	верхней панелью
	•	фильтрами
	•	кнопками добавления товара
	•	таблицей/списком товаров
	•	ярлыками
	•	сортировками
	•	быстрыми действиями
	•	переходом в карточку товара
	•	переходом в раздел маркировки

⸻

2. Верхняя часть страницы

В верхней части разместить:

Заголовок:

Товары

Справа 2 основные кнопки:
	1.	Добавить товар вручную
	2.	Импортировать с WB

Кнопки визуально разделить:
	•	основная чёрная кнопка — Добавить товар вручную
	•	вторичная/фиолетовая или светлая — Импортировать с WB

⸻

3. Фильтры и поиск

Под заголовком сделать панель фильтров:
	•	поиск по названию товара
	•	поиск по артикулу
	•	фильтр по категории
	•	фильтр по цвету
	•	фильтр по наличию:
	•	все
	•	есть в наличии
	•	нет в наличии
	•	фильтр по признаку:
	•	импортирован с WB
	•	создан вручную
	•	фильтр по маркировке:
	•	ЧЗ нужен
	•	ЧЗ не нужен

Добавить сортировку:
	•	по дате добавления
	•	по дате обновления

С вариантами:
	•	сначала новые
	•	сначала старые

Важно:
если появился новый товар, он должен автоматически показываться первым в списке.

⸻

4. Блок работы с ярлыками

Добавить отдельный блок/фильтр по ярлыкам.

Ярлыки по умолчанию:
	•	Новинка
	•	В акции
	•	Продвигаем
	•	ТОП
	•	Сливаем

Логика:
	•	ярлыки можно назначать вручную
	•	можно создавать и другие пользовательские ярлыки вручную
	•	у каждого ярлыка свой цвет

Особая логика ярлыка «Новинка»:
	•	если товар новый, ярлык Новинка присваивается автоматически
	•	ярлык горит 14 дней
	•	товар с ярлыком Новинка должен показываться в списке эти 14 дней, даже если товара нет в наличии

Фильтр по ярлыкам:

Сделать выбор:
	•	включить товары с выбранными ярлыками
	•	исключить товары с выбранными ярлыками

Рядом с каждым ярлыком показать:
	•	+ количество товаров с этим ярлыком
	•	– исключить из выборки

Пример:
	•	Новинка +12
	•	В акции +8
	•	Сливаем +3

Можно выбрать сразу несколько ярлыков.

⸻

5. Список товаров

Сделать список в виде удобной таблицы/строк.

Колонки:
	•	фото товара
	•	название товара
	•	артикул
	•	цвет
	•	размеры
	•	количество
	•	ярлыки
	•	признак WB
	•	действия

В строке товара показать:
	•	миниатюру фото
	•	полное название
	•	артикул
	•	цвет
	•	список размеров
	•	остаток
	•	ярлыки
	•	отметку, если товар импортирован с WB
	•	иконки действий

Для товаров, импортированных с WB:

добавить заметный индикатор:
	•	маленький бейдж WB
или
	•	подпись Импорт с WB

⸻

6. Поведение при клике

Клик по названию товара

При клике на название товара открывается страница редактирования карточки товара.

Клик по размеру

При клике на размер открывается маленький popover / dropdown / tooltip с данными:
	•	размер
	•	баркод
	•	иконка копирования баркода
	•	количество КИЗов по этому размеру

Пример:
	•	Размер: 46
	•	Баркод: 2039…
	•	Кнопка: копировать
	•	КИЗов: 22

⸻

7. Карточка товара

При переходе в карточку товара показать:
	•	фото товара
	•	название
	•	артикул товара
	•	артикул продавца
	•	бренд
	•	цвет
	•	категория
	•	состав
	•	список размеров
	•	баркоды
	•	остатки
	•	количество КИЗов
	•	применённые ярлыки
	•	признак WB
	•	кнопка Маркировка
	•	кнопка Открыть на WB если товар импортирован с Wildberries

⸻

8. Раздел «Маркировка»

Из карточки товара сделать кнопку перехода на страницу Маркировка.

На странице маркировки показать:

Таблицу по размерам:
	•	баркод
	•	рос. размер
	•	размер
	•	ЧЗ / количество КИЗов
	•	действие печати

Блок печати:
	•	выбор шаблона этикетки
	•	отправка на печать
	•	предпросмотр этикетки
	•	выбор количества
	•	выбор принтера

Шаблоны печати:
	•	ШК
	•	ЧЗ
	•	Совмещённое
	•	другие варианты при необходимости

Блок принтера:
	•	выбрать принтер
	•	редактировать принтер
	•	вместимость
	•	порог предупреждения
	•	остаток марок/ресурса

Важное:

На странице маркировки должна быть возможность:
	•	выбрать шаблон печати
	•	отправить этикетку на печать
	•	редактировать настройки принтера

⸻

9. Действия в списке товаров

Для каждой строки товара предусмотреть иконки:
	•	редактировать
	•	удалить
	•	открыть маркировку
	•	добавить/редактировать ярлык

⸻

10. UX-требования

Сделать интерфейс:
	•	чистым
	•	быстрым
	•	без перегруза
	•	удобным для работы с большим количеством SKU

Важно продумать:
	•	фиксированную шапку таблицы
	•	понятные hover-состояния
	•	компактные теги
	•	быстрые фильтры
	•	удобный popover по размерам
	•	акцент на остатках, ярлыках и маркировке

⸻

11. Визуальный стиль

Стиль:
	•	как современный SaaS
	•	минимализм
	•	мягкие серые фоны
	•	чёрные основные кнопки
	•	фиолетовые акценты для WB/ярлыков/статусов
	•	скругления 12–16 px
	•	аккуратные карточки и таблицы
	•	чистая типографика без перегруза

⸻

12. Что нужно отрисовать в Figma

Сделать экраны:
	1.	Страница списка товаров
	2.	Список товаров с фильтрами по ярлыкам
	3.	Popover по клику на размер
	4.	Карточка товара / редактирование
	5.	Страница Маркировка
	6.	Модалка печати этикетки
	7.	Модалка редактирования принтера
