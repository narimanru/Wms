ТЗ для Figma Make

Шаг 3 — Подтверждение группы товаров

Цель

Позволить пользователю:
	•	подтвердить автоматически найденный товар
	•	добавить связанные товары
	•	сформировать группу товаров, между которыми будут распределяться коды маркировки

Этот шаг готовит данные для шага «Распределение кодов».
