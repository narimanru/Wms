# 🛍️ Шаг 3: Подтверждение группы товаров

## 📋 Обзор

Шаг 3 в wizard'е "Импорт кодов маркировки" позволяет:
- ✅ Подтвердить автоматически найденный товар
- ✅ Добавить связанные товары в группу
- ✅ Просмотреть доступные размеры для каждого товара
- ✅ Сформировать группу для распределения КИЗов

---

## 🎯 Цель шага

Подготовить **группу товаров**, между которыми будут распределяться коды маркировки на следующих шагах. Размеры отображаются для информации.

## 🔒 Правила добавления товаров

### Условия добавления:

✅ **Можно добавить**, если совпадает:
- Категория с найденным товаром
- Цвет с найденным товаром

❌ **Нельзя добавить**, если:
- Товар уже в группе
- Категория отличается
- Цвет отличается

### Сообщения об ошибках:

**Несовпадение категории/цвета:**
```
Можно добавлять только связанные товары 
с одинаковой категорией и цветом.
```

**Дубликат:**
```
Этот товар уже добавлен в группу.
```

```typescript
onClick={() => setShowAddProductModal(true)}
```
- Открывает модальное окно
- Фильтрует уже добавленные товары

### Удаление товара из группы

```typescript
onClick={() => removeProductFromGroup(productId)}
```
- Удаляет товар из массива productGroup
- Обновляет счетчик

### Выбор размера

```typescript
onChange={() => toggleSizeSelection(productId, size)}
```
- Добавляет/удаляет размер из selectedSizes
- Обновляет визуальное состояние
